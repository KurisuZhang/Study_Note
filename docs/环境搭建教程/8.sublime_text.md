# sublime text 3

## 快捷键
1. ctrl + =  增大字体
2. ctrl + shift + p  打开控制面版
  - 输入“Install”，然后选择“Package Control: Install Package”
3. home 句首  end句尾 （mac: control + e）
4. Ctrl+L 选中当前行
